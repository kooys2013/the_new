# 어체 및 경어 사용 가이드

## 한국어 어체 원칙

### ✅ 권장 표현
| 상황 | 권장 표현 | 지양 표현 |
|------|-----------|-----------|
| 인사 | 수고많으십니다 | 안녕하세요 (사내 공지엔 과도하게 캐주얼) |
| 요청 | ~부탁드립니다 / ~주시기 바랍니다 | ~해주세요 (너무 구어체) |
| 안내 | ~하고자 합니다 / ~예정입니다 | ~할 거예요 |
| 마무리 | 이상입니다. 수고하십시오. | 감사합니다 (공지 끝맺음으로 어색) |
| 참고 | ※ 기호 활용 | 별(*) 또는 - 기호 |

### 어투 기준
- 격식체 `~입니다 / ~습니다` 유지
- 과도한 겸양어 지양: `~드리고자 하는 바입니다` (X) → `~하고자 합니다` (O)
- 번호 목록 후 마무리: `등을 기대하고 있습니다.` / `등을 목표로 합니다.`
- 요청 강도: 협조 요청 시 → `~부탁드립니다`, 필수 사항 → `~주시기 바랍니다`

## 일본어 어체 원칙

### ✅ 권장 표현
| 상황 | 권장 표현 |
|------|-----------|
| 인사 | いつもお世話になっております。 |
| 요청 | ～をお願いいたします / ～いただけますでしょうか |
| 안내 | ～する予定です / ～いたします |
| 마무리 | 以上です。よろしくお願いいたします。 |
| 주의사항 | ※ 기호 동일 사용 |

### 피해야 할 표현
- `です・ます` 체를 끝까지 유지 (혼용 금지)
- `～させていただきます` 남용 주의 (2회 이상 반복 시 어색)
- 한국어 원문에서 「수고하십시오」→ `よろしくお願いいたします`로 변환 (직역 금지)

## 공통 형식 규칙

- 시스템/제품명: `「 」` 꺾쇠 인용부호 (한/일 공통)
- 날짜: `3/16(월)` 또는 `3月16日(月)` 형식
- 기간: `○/○ ~ ○/○ (약 ○주간)` 형식
- 접속 링크 배포 같은 세부 일정: Tab 들여쓰기로 구분
- 영문 약어: Feedback, Teams, Form 등은 영문 그대로 유지
