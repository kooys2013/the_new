# KYT 위험예지훈련 시트 자동 생성 스킬

## 개요
사용자가 제공한 **작업 주제 + 현장 사진**을 분석하여
오프닝 안전 안내 멘트 + 5라운드 KYT 시트를 완성된 형태로 생성한다.

---

## 입력 처리

### 입력 유형별 처리
| 입력 | 처리 방법 |
|------|-----------|
| 주제(텍스트) + 사진(이미지) | 사진 속 작업 환경·장비·자세를 시각 분석하여 주제와 결합 |
| 주제(텍스트)만 | 해당 작업의 일반적인 위험요인을 전문 지식으로 생성 |
| 사진만 | 사진에서 작업 유형을 추론하여 주제 자동 설정 후 생성 |

### 사진 분석 시 착안사항
사진이 제공된 경우 아래를 반드시 확인하고 KYT에 반영:
- 작업자의 자세 및 보호구 착용 여부
- 주변 설비·배관·전기 설비의 상태
- 바닥 상태 (습기, 요철, 장애물)
- 고소 작업 여부, 중량물 여부
- 밀폐 공간, 고온·고압 환경 여부

## 출력 구성

### [오프닝 안전 안내 멘트]
아래 형식으로 사회자/팀장이 낭독할 수 있는 멘트를 생성:

```
📢 오늘의 KYT를 시작하겠습니다.

오늘 우리가 다룰 작업은 [작업명]입니다.
이 작업에서는 [주요 위험 요소 2~3가지]와 같은 위험이 
잠재되어 있습니다.

오늘 KYT를 통해 이러한 위험을 사전에 파악하고, 
구체적인 안전 대책을 함께 도출하여
무재해 작업을 실현하겠습니다.

모두 집중해 주시기 바랍니다.
```

---

### 1라운드 — 현상 파악
**작업 복잡도 기준으로 위험요인 개수 자동 결정:**
- 단순 작업 (1개 공정, 저위험): 3~4개
- 일반 작업 (복수 공정, 중위험): 4~5개  
- 복합·고위험 작업 (밀폐·고소·전기·화학 등): 5~7개

**작성 형식:**
```
1. [구체적 상황]해서 [부상/사고 결과]하게 된다.
2. ...
```

위험요인 작성 기준:
- 반드시 "~해서 ~하게 된다" 형식 준수
- 추상적 표현 금지 → 실제 현장 언어로 구체화
- 사진이 있는 경우 사진 속 상황을 직접 반영

---

### 2라운드 — 본질 추구
1라운드 위험요인 중 **가장 치명적·빈발 가능성 높은 1개** 선정

```
☆ 위험 포인트: [번호. 항목 전체]
→ 선정 이유: [한 줄 이유 — 재해 심각도 또는 발생 가능성 근거]
```

---

### 3라운드 — 대책 수립
2라운드 위험 포인트에 대한 **실행 가능한 구체적 대책** 3~4개

```
1. [확인/점검 대상]을 확인하여 [기대 효과]한다.
2. [보호구/안전장비]을 착용하여 [보호 효과]한다.
3. [절차/규칙]을 준수하여 [사고 방지]한다.
(필요 시 4번 추가)
```

---

### 4라운드 — 목표 설정
팀 전체가 함께 외치는 행동 목표 선언문

```
→ "○○을 철저히 하여 ○○사고 제로를 달성한다!"
```

---

### 5라운드 — 원 포인트 안전구호
4라운드 목표를 **5~8자 이내**로 압축한 구호 (3회 제창용)

```
"○○! 안전 확인! 전원 OK!"
(또는 작업 특성에 맞는 간결한 구호)
```
→ 3회 제창 지시: "(구호를 3회 함께 외칩니다)"

---

## 품질 기준
- 모든 위험요인은 해당 업종·작업의 실제 재해 사례를 기반으로 작성
- 대책은 "착용", "확인", "실시" 등 동사로 마무리하여 실행 가능성 강조
- 전문 용어와 현장 언어의 균형 유지
- KYT 교육 현장에서 바로 사용할 수 있는 완성도 목표

## 산업안전 전문 지식 베이스 (참고)

### 고위험 작업 유형별 주요 위험요인

**고소작업 (2m 이상)**
- 추락, 낙하물, 발판 불안정, 안전대 미착용

**전기작업**
- 감전, 아크 플래시, 충전부 접촉, 잔류전하

**밀폐공간**
- 산소결핍, 유해가스, 구조 지연

**화학물질 취급**
- 피부/눈 접촉, 흡입, 화재·폭발

**중량물 취급**
- 협착, 요통, 낙하, 불균형

**회전체/기계 주변**
- 말림, 끼임, 파편 비산

**용접·절단**
- 화재, 자외선 노출, 흄 흡입, 감전
